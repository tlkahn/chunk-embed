BGE-M3 Embedding Model

The BGE-M3 model (~2 GB) is required for generating embeddings.

  • Automatic: The model is downloaded automatically the first time you run an embed operation. No action needed.
  • Manual: Clone or copy the model into the local_bge_m3/ directory at the project root: 
    git clone https://huggingface.co/BAAI/bge-m3 local_bge_m3
