Editor ($VISUAL / $EDITOR)

An editor is needed to open source files from search results at the correct line number.

Set one of these environment variables in your shell profile (~/.zshrc, ~/.bashrc, ~/.config/fish/config.fish, etc.):

# Pick one:
export EDITOR=code      # VS Code
export EDITOR=nvim      # Neovim
export EDITOR=vim       # Vim
export EDITOR=emacs     # Emacs
export EDITOR=subl      # Sublime Text

For GUI editors that detach from the terminal, set $VISUAL instead (it takes priority over $EDITOR):

export VISUAL="code --wait"

After editing, reload your shell or restart the app.
