PostgreSQL + pgvector

A PostgreSQL database with the pgvector extension is required for storing and searching embeddings.

1. Install PostgreSQL

# macOS (Homebrew)
brew install postgresql@17
brew services start postgresql@17

# Ubuntu / Debian
sudo apt install postgresql postgresql-contrib

2. Install pgvector

# macOS (Homebrew)
brew install pgvector

# From source (any platform)
git clone https://github.com/pgvector/pgvector.git
cd pgvector
make && sudo make install

3. Create the database and enable the extension

createdb chunk_embed
psql chunk_embed -c "CREATE EXTENSION IF NOT EXISTS vector;"

4. Verify

Use the Test Connection button on the Setup tab to confirm everything works. The default connection URL is:

postgresql://localhost/chunk_embed

Set the DATABASE_URL environment variable to use a different database.
